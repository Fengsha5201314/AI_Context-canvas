# 📋 复制功能和自动捕获使用指南

## 🎉 新功能概览

我们已经成功实现了两个核心功能：

1. **快速复制功能**：选中卡片并复制到其他地方
2. **自动捕获剪贴板**：自动监听并添加复制的内容

---

## ✅ 功能 1：快速复制

### 如何使用

#### 1. 选中卡片

**单选**：
- 直接点击一个卡片
- 卡片会显示蓝色边框和高亮效果

**多选**：
- 按住 `Ctrl`（Windows）或 `Cmd`（Mac）
- 点击多个卡片
- 所有选中的卡片都会高亮显示

**全选**：
- 方法 1：按 `Ctrl+A`（Windows）或 `Cmd+A`（Mac）
- 方法 2：点击工具栏的 **"全选"** 按钮

#### 2. 复制内容

**方法 1：键盘快捷键**
- 选中卡片后，按 `Ctrl+C`（Windows）或 `Cmd+C`（Mac）
- 会弹出提示：已复制 X 个卡片到剪贴板

**方法 2：工具栏按钮**
- 选中卡片后，点击工具栏的 **"复制选中"** 按钮
- 会弹出提示：已复制 X 个卡片到剪贴板

#### 3. 粘贴到其他地方

- 打开 AI 对话框（ChatGPT、Claude、Kimi 等）
- 按 `Ctrl+V`（Windows）或 `Cmd+V`（Mac）
- 内容会被粘贴到对话框

### 复制格式

复制的内容支持多种格式：

- **纯文本**：文字内容会以纯文本形式复制
- **富文本（HTML）**：保留格式的文本
- **图片**：如果只选中一张图片，会直接复制图片

### 视觉反馈

- **选中状态**：蓝色边框 + 蓝色高亮圈
- **选中数量**：工具栏显示 "已选中 X 个"
- **按钮状态**：
  - 没有选中时，"复制选中" 按钮为灰色（不可用）
  - 选中后，按钮变为蓝色（可用）

## 🔄 功能 2：自动捕获剪贴板

### 如何使用

#### 1. 开启自动捕获

在工具栏找到 **"自动捕获剪贴板"** 复选框，勾选它。

**开启后的标志**：
- 复选框被勾选
- 旁边有一个绿色的闪烁圆点 🟢

#### 2. 正常使用

开启后，你可以正常与 AI 对话：

1. **截图**（QQ、微信、飞书等）
2. **复制文字**（任何地方）
3. **切换到浏览器窗口**
4. 工具会自动检测剪贴板变化
5. 新内容自动添加到画布

### 工作原理

- 每 500ms 检查一次剪贴板
- 检测到新内容时自动添加
- 防止重复添加相同内容
- 只在浏览器窗口激活时工作

### 使用场景

**场景 1：与 AI 对话时收集素材**
```
1. 打开工具，开启自动捕获
2. 与 ChatGPT 对话
3. 复制有用的回答
4. 切换到工具窗口
5. 内容自动添加
6. 继续与 AI 对话
7. 重复步骤 3-6
```

**场景 2：收集多个来源的内容**
```
1. 打开工具，开启自动捕获
2. 在网页 A 复制文字
3. 在网页 B 截图
4. 在微信复制消息
5. 切换到工具窗口
6. 所有内容都已自动添加
```

### 注意事项

⚠️ **需要浏览器窗口激活**
- 自动捕获只在浏览器窗口激活时工作
- 如果你在其他应用复制内容，需要切换回浏览器

⚠️ **防重复机制**
- 相同的内容不会重复添加
- 如果你多次复制相同内容，只会添加一次

⚠️ **权限要求**
- 首次使用时，浏览器可能会请求剪贴板权限
- 请点击"允许"以启用自动捕获功能

## 🐛 常见问题

### Q1：为什么自动捕获不工作？

**可能原因：**
1. 没有开启"自动捕获剪贴板"开关
2. 浏览器窗口没有激活
3. 浏览器没有剪贴板权限

**解决方法：**
1. 检查工具栏的开关是否勾选
2. 切换到浏览器窗口
3. 刷新页面，允许剪贴板权限

### Q2：为什么复制后粘贴不了？

**可能原因：**
1. 没有选中卡片
2. 浏览器不支持剪贴板 API

**解决方法：**
1. 确保卡片有蓝色高亮边框
2. 使用最新版 Chrome/Edge 浏览器

### Q3：为什么内容重复添加？

**可能原因：**
- 防重复机制失效

**解决方法：**
- 手动删除重复的卡片
- 关闭并重新开启自动捕获

### Q4：为什么图片复制不了？

**可能原因：**
- 浏览器限制

**解决方法：**
- 使用"导出PDF"或"导出Word"功能
- 或者使用截图工具重新截图
